## 程序运行：  
./cmake-build-debug/myPano2anyview ./data/image ../data/front_image ../data/top_image  

## 数据说明：  
├── image //原始图像  
├── image.txt //原始图像列表  
├── front_image //转换后的前视图像  
└── top_image //转换后的俯视图图像
